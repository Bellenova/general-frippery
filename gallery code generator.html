How to use this form

  1. Select the number of images you would like to include in your gallery from the drop down below.
  2. Enter the thumbnail src, full size src, and caption in the appropriate text field for each image. If you do not have a thumbnail, enter the full size src in both the full size and thumbnail field.
  3. Click the "Generate your code" button.
  4. Your code will appear in the text area below. Copy the content in the text area and paste it in the html editor of the page you are editing.
